# Patrick AI Dental Workflow Safety Portfolio

Public static portfolio for Patrick Chen, a dentist-builder focused on local-first, synthetic-first, human-review-first workflow safety infrastructure for dental and healthcare-adjacent systems.

This portfolio is about the layer before healthcare-adjacent AI reaches clinical output: structured context, provenance checks, review harnesses, risk labels, export boundaries, and human review.

## Public Boundary

This site is a static portfolio. It does not diagnose, recommend treatment, interpret images, provide prognosis, provide teledentistry, provide patient-facing automation, claim clinical validation, claim regulatory clearance, or operate on real patient records.

All examples are synthetic portfolio examples only. They are not medical advice and are not for real patient use.
