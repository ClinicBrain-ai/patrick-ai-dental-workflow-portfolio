# Patrick Chen AI Dental Career & Migration Portfolio

This repository is a local-only static portfolio foundation for Patrick Chen's English AI dental and healthcare workflow safety portfolio.

Positioning:

> Dentist building safe, human-in-the-loop AI workflow infrastructure for dental and healthcare-adjacent use cases.

The site focuses on non-diagnostic workflow design, synthetic-data-first prototypes, provenance-aware documentation, explicit risk boundaries, and manual review for sensitive outputs.

## Local-Only Status

This project is static HTML and CSS only. It is intended to run locally by opening `index.html` in a browser.

No backend, analytics, tracking, forms, external APIs, deployment configuration, or GitHub Actions deployment setup is included.

## How To Open Locally

Open `index.html` directly in your browser.

Optional from the repository folder:

```sh
open index.html
```
