Clinical AI Review Harness

Workflow Safety Review Harness for Synthetic Drafts.

A local-only, synthetic-data-first safety evaluation concept for checking whether AI-generated summaries, drafts, and review items stay inside safe boundaries.

This harness evaluates workflow-safety boundaries in synthetic drafts. It is not a medical benchmark, not clinical validation, and not a diagnostic evaluation system.

Project Detail

Problem

AI-generated workflow drafts can drift into unsupported claims, final-sounding advice, or unclear source attribution if there is no review harness.

Why this matters

Healthcare-adjacent AI workflows need explicit checks for scope, provenance, and human-review requirements before drafts are used.

What this project does

It models checks for safe routing, prohibited-output detection, provenance completeness, and required human review using fictional examples.

What this project does NOT do

It does not claim clinical validation, benchmark performance, medical accuracy, diagnosis capability, treatment recommendation capability, prognosis, or image interpretation.

Current prototype status

Static portfolio concept with example result tables and boundary categories; no runtime evaluator and no external service integration.

Safety Boundary

The harness checks workflow language and documentation structure only. It does not evaluate clinical correctness or replace professional judgment.

Human-In-The-Loop Workflow

Every flagged draft routes to a human reviewer for revision, escalation, or rejection. Passing a boundary check is not approval for patient-facing final advice.

Portfolio Value

This project demonstrates Patrick's safety-first approach to AI workflow infrastructure, including explicit failure modes and review gates.

Public Evidence Boundary

Detailed rubrics, synthetic JSON results, diagrams, and rewrite files are excluded from active public navigation until separately audited. This approved page keeps the public evidence focused on the review-harness concept and its human-review boundary.
