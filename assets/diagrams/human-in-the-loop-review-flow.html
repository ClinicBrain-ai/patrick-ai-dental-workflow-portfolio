Human-In-The-Loop Review Flow

Static synthetic diagram for manual review boundaries in AI-assisted dental workflow prototypes.

1. Draft

Non-final output

AI-assisted workflow text is created as a draft only.

2. Screen

Boundary check

Look for prohibited claims, missing sources, and privacy risk.

3. Review

Human judgment

A human reviewer edits, escalates, rejects, or archives.

4. Approve

Manual approval

Approval means manual handling only, not automatic delivery.

5. Record

Audit note

State changes and reviewer notes remain reviewable.

Stop condition: any diagnosis, treatment recommendation, prognosis, image interpretation, automatic second opinion, real PHI, or final patient-facing message must be rejected or escalated.
