# Static Public Deployment Post-Deployment Checklist

Use this checklist only after a separate manual static deployment has been performed.

This checklist does not deploy the site.

## Page Checks

- [ ] Open homepage.
- [ ] Check navigation.
- [ ] Check Profile.
- [ ] Check Walkthrough.
- [ ] Check Projects.
- [ ] Check Safety.
- [ ] Check Contact.

## Boundary Checks

- [ ] Confirm no form appears.
- [ ] Confirm no patient intake appears.
- [ ] Confirm no real contact details appear.
- [ ] Confirm no analytics were added.
- [ ] Confirm no external scripts load.
- [ ] Confirm disclaimers still visible.
- [ ] Confirm Taiwan Dental Prep remains non-service framing.
- [ ] Confirm no clinical claims appear.
- [ ] Confirm public URL is shared only after manual human review.

## Technical Checks

- [ ] Confirm no JavaScript was added.
- [ ] Confirm no backend was added.
- [ ] Confirm no database was added.
- [ ] Confirm no tracking automation was added.
- [ ] Confirm no external integrations were added.
- [ ] Confirm no contact capture was added.

## Data Checks

- [ ] Confirm no real PHI appears.
- [ ] Confirm no real patient data appears.
- [ ] Confirm no real imaging files appear.
- [ ] Confirm no real clinic details appear.
- [ ] Confirm no real reviewer identities or contact details appear.

## Status

- [ ] Record `[Public URL After Manual Deployment]` only after review.
- [ ] Record whether deployment remains static-only.
- [ ] Record any issue found before sharing the public URL.
